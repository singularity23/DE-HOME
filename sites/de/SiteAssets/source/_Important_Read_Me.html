Intro

In order to modify the main page, you have to modify the corresponding files under this directory. Download all .md files. Open the one you need to edit - each .md file is related to each panel on the HydroShare.

For example, if you want to edit the link under "Administrative", then you need to open "1_Administrative.txt" file with Notepad.

Update the .txt Files

In general, each entry consists of eight inputs, separated by comma:

Indent Level(1), Title(2), Tag(3), Date(4), Link(5), Icon(6), Icon Source(7), Icon Link(8)

1. Indent Level: use 1 for the first level indent, use 2 for second level indent.

2. Title: The description of the link you are going to add

3. Tag: This is optional - but you have two entries to choose new or update

4. Date: format: yyyy-mm-dd used together with the new or update input, keep it empty if the last input is empty

5. Link: The url link of the document/directory you want to point to - J drive link works as well

6. Icon, 7. Icon Source, 8. Icon Link: These three need to be used in conjunction, you won't need them for most of the time

Lists and Links:

List item example in html:

  • E13 Desk Booking

1, E13 Desk Booking, , , http://pq.bchydro.bc.ca:99/index.php, , , ,

Indented list item example in html:

  • DELT
    • Business Review

1, DELT(Distribution Engineering Leadership Team), , , /sites/de/sites/de/DELT/Forms/AllItems.aspx, , , ,
    

  2, Business Review, new, "yyyy-mm-dd", http://pq.bchydro.bc.ca:99/index.php, , , ,

Abbreviations:

In the above example:

( ) is used behind the abbreviation DELT, to indicate the full name when your mouse hover over the abbreviation.

New and Updated Items:

For new link item or updated link item:

This is optional, however, the tag (column 3) and date (column 4) need to be used in conjunction.

Make HTML Code

Once you have edited the .md files, click Here. Choose the all .txt files that you just downloaded and edited, then click on Submit. The html code will be generated. You can preview the results from there. Click on Copy HTML Code, then paste it to HydroShare to replace all the existing code.
