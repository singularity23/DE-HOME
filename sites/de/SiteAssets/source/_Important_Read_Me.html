Intro

In order to modify the main page, you have to modify the corresponding files under this directory. Download all .md files. Open the one you need to edit - each .md file is related to each panel on the HydroShare.

For example, if you want to edit the link under "Administrative", then you need to open "1_Administrative.md" file with Notepad.

Update the .md Files

Lists and Links:

List item example in html:

  • E13 Desk Booking

in .md file:

* [E13 Desk Booking](http://pq.bchydro.bc.ca:99/index.php)

Indented list item example in html:

  • DELT
    • Business Review

in .md files:


        
* [<abbr
            title="Distribution Engineering
            Leadership Team">DELT</abbr>](/sites/de/sites/de/DELT/Forms/AllItems.aspx)

        

           ** [Business
            Review<new></new>](/sites/de/Business_Review/Forms/AllItems.aspx)

[ ] contains the text description of your choice for the link; ( ) includes the actual url of the link, e.g. https://, file:\\\,etc.;

For indented list, adding either a space ' ' or a star * works

Note: If the link is from network drive, like J:\\Engineering\Distribution\CHARGE CODES\F24\ This is not a properly decoded format. Paste this link into your broswer, and press enter. Copy the new link in the search bar. It should look like this: (file:///)J://Engineering/Distribution/CHARGE%20CODES/F24/ with and without file:/// are both fine

Abbreviations:

In the above example:

[ ] other than DELT, it is wrapped around with <abbr title="Distribution Engineering Leadership Team">DELT</abbr>. This is to show the full name of this abbriviation in html.

New and Updated Items:

For new link item or updated link item:

Inside [ ], add tags <new></new> in the end for new link items, add <update></update> for updated link items. This is not conventional mark down syntax, this is for html rendering purpose

Make HTML Code

Once you have edited the .md files, click Here. Choose the all .md files that you just downloaded and edited, then click on Submit. The html code will be generated. You can preview the results from there. Click on Copy HTML Code, then paste it to HydroShare to replace all the existing code.
