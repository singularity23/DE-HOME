Intro

All links and information shown on DE hydroshare are here. They are organized in a JSON file format.

Download this file: WebLinks.txt.

1. Open the file:

Use any text editor if you are familiar with JSON file format; if not, use this JSON Online Editor, in "Tree Mode" and "Expand All".

2. Understand the structure:

Use an example, information are shown in the txt file like this:

{
  "name": "Organization & Teams",
  "subheaders": [
    {
      "title": "Regional Teams",
      "links": [
        {
          "name": "Lower Mainland North",
          "url": "",
          "info": "",
          "sub_links": [
            {
              "url": "https://hydroshare.bchydro.bc.ca/sites/de/Lists/SharedDocuments/DE%20Photo%20Org%20Chart/LMN.pdf",
              "name": "Member Details"
            },
            {
              "url": "https://hydroshare.bchydro.bc.ca/sites/de/Lists/SharedDocuments/Regional%20Maps/LMN%20-%20Regional%20Area%20Substations.pdf",
              "name": "Substation Assignments"
            }
          ]
        }
      ]
    }
  ]
},

They will be renderred in hydroShare like this:

Organization & Teams
Regional Teams
Lower Mainland North Member Details   Substation Assignments  

In general, each element follows this pattern:

{
    "name": "Link Name",
    "url": "https://example.com",
    "info": "Short description",
    "sub_links": [
      {
        "name": "Optional sub‑item",
        "url": "https://example2.com"
        "class": "Optional for handling special files"
      }
    ]
  }

"Class" under each "Sub_link" is optional and is used for handling special files, "Class":"_form" for files need to be downloaded; "Class":"_clipboard" for files that are copied to clipboard.

3. Format and Validation:

The online editor will keep the format and validate the JSON structure of each item continously.

4. Save and Upload:

Save the file and upload and replace the file wherever it is downloaded from.

Once it is uploaded, the sharepoint site will update automatically
