Aspen Query Tool - Beta

Background

This tool is intended to simplify the process of collecting station protection settings from Aspen. Any data shown is 100% up to date with Aspen.

Installation

This tool has only been tested in Chrome.
  1. Drag and drop this link into your bookmarks bar: Aspen Query
  2. Also drag and drop this link into your bookmarks bar: Aspen Table Format

Using the tool

  1. Click the bookmark "Aspen Query"
  2. Select the desired substation and feeder and hit submit
  3. If the relay is electro-mechanical, click the bookmark "Aspen Table Format" to nicely format the results
